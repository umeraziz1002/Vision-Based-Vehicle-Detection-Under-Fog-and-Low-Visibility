About This Project

Final Year Project — BS Computer Science

Vision-Based Vehicle Detection
under Fog and Low Visibility Conditions

This project addresses one of the most critical challenges in intelligent transportation systems — detecting vehicles accurately when visibility is severely reduced due to fog, haze, rain, or night conditions.

Project Objectives
  • Develop a robust vehicle detection system using YOLOv8 deep learning
  • Handle foggy and low-visibility road environments effectively
  • Provide a user-friendly web interface for real-time detection
  • Support image, video, and live webcam detection modes
  • Expose REST API endpoints for future mobile app integration
